# 12 main modules of the project





1. ### Project Scaffolding \& Config — monorepo setup, env, DB, Redis, logging, CORS, security middleware
2. ### Authentication \& RBAC — JWT access/refresh tokens, role guards, protected routes
3. ### User Management — CRUD, role assignment, activation/deactivation
4. ### Client Management — company profiles, SLA tiers, contact persons, client portal access
5. ### Vehicle Management — fleet registry, compliance tracking, odometer logs, document uploads
6. ### Driver Management — profiles, license expiry tracking, vehicle assignment history
7. ### Service Request Workflow — full lifecycle, status engine, priority, notes, attachments, cost tracking
8. ### Maintenance Ledger — records, recurring templates, cost analytics, vehicle timeline
9. ### Scheduling \& Calendar — appointments, technician availability, workshop capacity
10. ### Communication Logs — manual channel logging, follow-up tracking
11. ### Notifications \& Automations — SSE, in-app alerts, all 8 cron jobs
12. ### Reports \& Audit Trail — 5 report types, PDF/CSV export, immutable audit log
